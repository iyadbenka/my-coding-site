مقدمة في HTML

HTML هي لغة بناء صفحات الويب. تستخدم لكتابة النصوص، الصور، والعناصر الأخرى على الموقع.

<h1>هذا عنوان</h1>

مقدمة في CSS

CSS تستخدم لتزيين صفحات الويب مثل تغيير الألوان، الخطوط، وتصميم الشكل العام.

body { background-color: lightblue; }

مقدمة في JavaScript

JavaScript تجعل الموقع تفاعلي. يمكنها تنفيذ الأوامر عند ضغط الزر أو تحريك الفأرة.

alert("مرحبا بك!");
